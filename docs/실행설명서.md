## 1. 사전 준비

### 필수 설치 항목

| 소프트웨어 | 확인 방법 | 다운로드 |
|-----------|-----------|----------|
| **Node.js** (v18 이상) | 명령 프롬프트에 `node -v` 입력 | https://nodejs.org |

`npm install` 은 **`start.bat` 실행 시 자동**으로 처리됩니다.

### 외부 접속용 추가 파일 (선택)

외부 친구와 플레이하려면 `cloudflared.exe` 가 필요합니다.
**이미 `chess_online` 폴더에 포함**되어 있습니다.
없을 경우 아래에서 다운로드 후 폴더에 넣으세요:

```
https://github.com/cloudflare/cloudflared/releases/latest
→ cloudflared-windows-amd64.exe 다운로드 후 이름을 cloudflared.exe 로 변경
```

## 2. 같은 WiFi에서 플레이 (로컬 모드)

같은 공유기에 연결된 기기끼리 플레이할 때 사용합니다.

### 실행

`start.bat` 더블클릭

```
========================================
        보드게임 온라인 서버 실행 중
========================================
  로컬:   http://localhost:3000
  LAN:    http://192.168.1.16:3000
  ↑ 같은 WiFi에서만 접속 가능 (외부 공개는 start-public.bat)
========================================
  종료: Ctrl+C 또는 stop.bat 실행
========================================
```

### 접속

- **이 컴퓨터에서**: 브라우저에 `http://localhost:3000` 입력
- **같은 WiFi의 다른 기기**: `LAN:` 에 표시된 주소 입력

## 3. 외부 친구와 플레이 (공개 모드)

수원, 제주, 미국 등 어디서든 접속 가능한 모드입니다.

### 동작 구조

```
start-public.bat
  └─ start-public.ps1 실행
       ├─ [1/2] cloudflared.exe 를 백그라운드로 실행
       │         → 터널 URL 자동 캡처 (최대 30초 대기)
       └─ [2/2] TUNNEL_URL 환경 변수와 함께 node server.js 실행
```

### 실행

`start-public.bat` 더블클릭

```
[1/2] Cloudflare 터널 시작 중... (최대 30초 대기)

========================================
  [OK] 터널 활성화 -- 아래 주소를 공유하세요

  >> https://random-words.trycloudflare.com <<

========================================

[2/2] Node.js 서버 시작...
```

### 공유 방법

1. `>> https://... <<` 사이에 표시된 주소를 복사
2. 카카오톡으로 친구에게 전송
3. 또는 관리자 페이지(`http://localhost:3000/admin.html`)에서 클릭 한 번으로 복사

> **주의**: `start-public.bat` 창을 닫으면 터널이 끊어져 외부 접속이 불가합니다.

## 4. 게임 방법 — 공통

### 방 만들기 (호스트)

1. 브라우저에서 서버 주소 접속
2. **게임 선택** (체스·오목·사목·오셀로·인디언 포커·체커 중 택 1)
3. **내 색상** 선택 (게임별 상이, 사목·인디언 포커는 고정)
4. **제한 시간** 선택:
   - 10분 / 30분: 각자 해당 시간 내에 모든 수를 완료
   - 무제한: 시간 제한 없음
   - 직접 설정: 1~120분 사이 원하는 시간 입력
5. **방 만들기** 클릭
6. 생성된 링크를 복사하여 친구에게 전송

### 방 참가 (게스트)

받은 링크를 클릭하면 자동으로 게임 시작

### 관전

방 번호를 알고 있으면 URL에 `?spectate=<방번호>` 를 추가하거나
로비에서 관전 버튼으로 입장 가능합니다.

### 게임 중 공통 기능

| 버튼 | 기능 |
|------|------|
| **기권** | 상대방 승리로 게임 종료 |
| **무승부 제안** | 상대방에게 무승부 제안 전송 (체스 전용) |
| **💬 채팅** | 채팅창 열기/닫기 (모바일) |
| **🔊 / 🔇** | 사운드 켜기/끄기 |

### 게임 종료 후

| 버튼 | 기능 |
|------|------|
| **재대국** | 같은 상대와 색상을 바꿔 다시 대국 |
| **새 게임** | 로비로 돌아가 새 방 생성 |

### 모바일 이용 시 주의사항

- iOS Safari: 주소창이 내려가는 현상이 있을 수 있습니다. 가로 모드 권장
- 채팅창을 열면 키보드가 올라와 화면이 좁아집니다. 채팅 후 닫기 버튼을 누르세요
- 체스 복기는 모바일에서도 지원합니다 (← → 스와이프 대신 버튼 사용)

## 5. 게임별 규칙

### ♟ 체스

- 이동할 기물을 **클릭** → 이동 가능한 칸 표시 → 목적지 클릭
- 폰이 상대 진영 끝에 도달하면 **승급 선택 화면** 자동 표시
- 게임 종료 후 **복기 모드** 진입 가능 (⏮◀▶⏭, 키보드 ← → 지원)

### ⬤ 오목 (렌주 룰)

- 빈 교차점을 클릭해 돌을 놓습니다
- **정확히 5개** 연결 시 승리 (6개 이상 장목은 무효)
- 흑이 선공

### 🔴 사목 (Connect Four)

- 상단 화살표 버튼 또는 열 클릭으로 돌 투하
- 중력이 적용됩니다 (빈 칸 있으면 아래로 떨어짐)
- **4개 연결** (가로·세로·대각선) 시 승리
- 호스트 = 빨강(선공), 게스트 = 노랑

### ⬜ 오셀로 (리버시)

- 상대 돌을 사이에 끼울 수 있는 칸에만 놓을 수 있습니다
- 유효 수 위치에 **점** 으로 표시됩니다
- 유효 수가 없으면 자동으로 **패스**됩니다
- 양측 모두 유효 수 없으면 돌 수 비교로 승패 결정
- 흑이 선공

### 🃏 인디언 포커

- 자기 카드는 이마에 대고 보지 않습니다 (화면에 "?" 로 표시)
- **상대방 카드**만 볼 수 있습니다
- 배팅 순서: 게스트 먼저 → 호스트 → 쇼다운
- 액션: **콜**(배팅 맞추기) / **레이즈**(5칩 추가, 최대 3회) / **폴드**(포기)
- **높은 숫자**가 이깁니다 (A < 2 < ... < K). 동점이면 호스트 승
- 칩이 앤티(5칩)보다 적으면 게임 종료
- 색상 선택 없음 (자동 배정)

### 🔴 체커 (Checkers)

- 어두운 칸에서만 이동합니다
- 기본 이동: 앞 대각선 1칸 이동
- **점프**: 상대 말을 대각선으로 뛰어넘어 잡을 수 있습니다
  - 점프 가능하면 **반드시 해야 합니다** (강제 점프 룰)
  - 연속 점프 가능하면 계속 같은 말로 이동합니다
- 상대 진영 끝에 도달하면 **킹**(♛) 으로 승격 → 전후 방향 이동 가능
- 상대 말을 전멸시키거나 이동 불가 상태로 만들면 승리
- 호스트: 빨강(선공), 게스트: 검정

## 6. 서버 종료

### 방법 1: stop.bat (권장)

`stop.bat` 더블클릭

```
[OK] 서버가 정상적으로 종료되었습니다.
```

### 방법 2: 관리자 페이지

`http://localhost:3000/admin.html` 접속 → **서버 종료** 버튼 클릭
(localhost 접속 시에만 버튼 활성화)

### 방법 3: Ctrl+C

서버 실행 창에서 `Ctrl+C`

## 7. 관리자 페이지

`http://localhost:3000/admin.html`

| 항목 | 설명 |
|------|------|
| 상태 점 (초록/빨간) | 서버 온라인/오프라인 여부 |
| 업타임 | 서버 실행 후 경과 시간 (HH:MM:SS) |
| 진행 중 | 현재 대국 중인 방 수 |
| 대기 방 | 게스트 입장을 기다리는 방 수 |
| 접속 중 | 현재 연결된 플레이어 수 |
| 전체 방 | 서버에 존재하는 총 방 수 |
| 외부 공개 주소 | Cloudflare 터널 URL (공개 모드 실행 시 표시) |
| 서버 종료 | localhost 접속 시 활성화 |

## 8. 문제 해결

### 서버가 시작되지 않는다

**증상**: `포트 3000가 이미 사용 중입니다` 메시지

**해결**:
1. `stop.bat` 실행으로 기존 서버 종료
2. 그래도 안 되면 작업 관리자에서 `node.exe` 프로세스 종료 후 재시작

---

### 친구가 접속이 안 된다 (공개 모드)

**확인 사항**:
1. `start-public.bat` 창이 열려 있는지 확인 (닫히면 터널 종료)
2. 터널 URL이 제대로 표시되었는지 확인
3. 새로 `start-public.bat` 을 실행하면 새 URL이 생성됨

---

### 사운드가 나지 않는다

- 게임 화면의 🔇 버튼 클릭하여 🔊 로 변경
- 첫 수를 두거나 버튼을 클릭하면 오디오 활성화 (브라우저 자동재생 정책)

---

### 체커에서 말이 움직이지 않는다

- 현재 **자신의 차례**인지 확인 (상단 상태 표시 확인)
- 점프 가능한 말이 있으면 그 말만 이동 가능합니다 (강제 점프)
- 연속 점프 중에는 같은 말만 계속 이동해야 합니다

---

### 인디언 포커에서 배팅 버튼이 안 보인다

- 자신의 배팅 차례가 아닐 때는 버튼이 숨겨집니다
- 상단에 "○○님 차례" 표시를 확인하세요
