## 개요

v1.2.0은 v1.0 이후 가장 큰 기능 업데이트입니다. **AI 혼자하기**, **보드 크기 선택**, **인디언 포커 룰 전면 개편**을 중심으로 사용자 경험과 게임 완성도를 크게 높였습니다.

## 신규 기능

### 1. 혼자하기 (vs AI)

6가지 게임 모두 AI를 상대로 혼자 연습할 수 있습니다.

| 게임 | AI 알고리즘 | 난이도 |
|------|------------|-------|
| 체스 | 미니맥스 depth-3 + alpha-beta pruning | 초급 |
| 오목 | 휴리스틱 패턴 매칭 (5목·열린4·막힌4·3 가중치) | 중급 |
| 사목 | 미니맥스 depth-6 + alpha-beta pruning | 중급 |
| 오셀로 | 미니맥스 depth-4 (구석·안정석 가중치) | 중급 |
| 체커 | 미니맥스 depth-4 + 강제 점프 | 중급 |
| 인디언 포커 | 카드 비교 기반 휴리스틱 + 블러프 확률 | 초급 |

**사용법**:
1. 로비에서 게임 카드의 `🤖 혼자` 버튼 클릭
2. 색상(선/후공) 선택
3. 오목·사목은 보드 크기도 선택 가능
4. 인디언 포커는 덱 수와 승리 조건 선택 가능
5. `시작` 클릭 → 즉시 AI 대국 시작

**특징**:
- 서버 연결 불필요 (완전 클라이언트 사이드)
- 체스·오목은 무르기(undo) 지원 (AI가 항상 수락)
- 솔로 대국 결과도 개인 전적에 자동 저장

### 2. 보드 크기 선택

오목과 사목에서 방 생성 시 보드 크기를 선택할 수 있습니다 (멀티플레이·솔로 모두 지원).

**오목 지원 크기**:
| 크기 | 설명 |
|------|------|
| 13×13 | 빠른 대국 |
| **15×15** | **기본 (렌주 공식 크기)** |
| 17×17 | 넓은 판 |
| 19×19 | 바둑판 크기 |

**사목 지원 크기**:
| 크기 | 설명 |
|------|------|
| 5행×4열 | 미니 |
| **6행×7열** | **기본 (공식 크기)** |
| 7행×8열 | 넓은 판 |
| 8행×9열 | 대형 |

### 3. 인디언 포커 룰 전면 개편

#### 카드 범위 변경
- **변경 전**: A ~ K (1~13, 총 52장/덱)
- **변경 후**: A ~ 10 (1~10, 총 10장/덱)

#### A(에이스) 특수 규칙
- A(1)는 **10을 상대로만 이깁니다**
- 나머지 경우(2~9 상대)에는 **최하위 카드**
- 예시: A vs 10 → A 승, A vs 9 → 9 승, A vs 2 → 2 승

#### 10 폴드 페널티
- 10을 가지고 있을 때 폴드하면 **앤티(5칩)만큼 추가 칩 손실**
- 게임 화면에 토스트 알림 표시
- AI도 동일한 규칙 적용 (AI는 10을 가질 때 절대 폴드하지 않음)

#### 덱 수 선택
- **1덱**: 10장 (빠른 게임)
- **2덱**: 20장 (기본)
- **3덱**: 30장 (긴 게임)

#### 승리 조건 선택
- **①칩 모두 획득**: 상대방 칩이 0이 되면 즉시 승리
- **②덱 소진 후 비교**: 덱 카드가 모두 소진되면 칩이 더 많은 쪽 승리 (기본)

### 4. 개인 전적 기록 (v1.1 포함)

- 게임별 승/패/무 통계를 브라우저에 저장
- 로비 우측 상단 `📊` 버튼으로 확인
- 기록 초기화 가능
- 게스트 프로필 (`👤`) 에서 닉네임 변경 가능 (최대 16자)
- 30일 비활동 시 전적 및 닉네임 자동 초기화

---

### 5. 게임 규칙 설명 (v1.1 포함)

- 각 게임 카드의 `📋 규칙` 버튼 클릭 → 규칙 모달 표시
- 게임 중에도 규칙 확인 가능
- 인디언 포커 규칙에 새 룰(A>10, 폴드 페널티, 덱 시스템) 반영

## 기술 변경사항

### 클라이언트 코드 분리

`game.js` 단일 파일에서 게임별 파일로 분리:

```
game.js              ← 공통 오케스트레이터 (소켓 이벤트, 공통 헬퍼)
game-chess.js        ← 체스 초기화 + 솔로 AI
game-omok.js         ← 오목 초기화 + 솔로 AI + 무르기
game-connect4.js     ← 사목 초기화 + 솔로 AI
game-othello.js      ← 오셀로 초기화 + 솔로 AI
game-checkers.js     ← 체커 초기화 + 솔로 AI
game-indianpoker.js  ← 인디언 포커 초기화 + 솔로 게임 루프
```

각 파일은 `window.GameHandlers[gameType]` 네임스페이스로 등록되어 `game.js`에서 동적으로 호출됩니다.

### AI 모듈

| 파일 | 내용 |
|------|------|
| `ai-chess.js` | chess.js 활용 미니맥스 |
| `ai-omok.js` | 패턴 점수 기반 휴리스틱 |
| `ai-connect4.js` | 미니맥스 depth-6 |
| `ai-othello.js` | 미니맥스 depth-4 |
| `ai-checkers.js` | 미니맥스 depth-4 |
| `ai-indianpoker.js` | 카드 비교 + 블러프 |

### URL 파라미터 (솔로 모드)

```
/game.html?solo=omok&color=black&boardSize=13
/game.html?solo=connect4&color=white&boardRows=7&boardCols=8
/game.html?solo=indianpoker&numDecks=2&winCondition=2
```

## 업그레이드 방법

### 로컬 개발 환경

```bash
git pull origin main
npm install   # 의존성 변경 없음 — 생략 가능
node server.js
```

### Render.com (자동)

`main` 브랜치에 push 시 Render.com이 자동으로 재배포합니다.
