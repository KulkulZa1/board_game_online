# 릴리즈 노트 — v1.0.0

> 릴리즈 날짜: 2026-03-28
> 배포 URL: https://board-game-online.onrender.com
> GitHub: https://github.com/KulkulZa1/board_game_online

---

## 개요

보드게임 온라인 **최초 공개 버전**입니다.

카카오톡 링크 하나로 친구를 초대하고 6가지 보드게임을 실시간으로 즐길 수 있습니다.
별도 설치·회원가입 없이 링크 클릭만으로 즉시 입장 가능합니다.

---

## 지원 게임 (6종)

| 게임 | 규칙 요약 |
|------|-----------|
| ♟ **체스** | chess.js 서버 검증, 폰 승급·캐슬링·앙파상 완전 지원 |
| ⬤ **오목** | 15×15 렌주 룰 — 정확히 5개 연결 승리 (장목 무효) |
| 🔴 **사목** | 7×6 중력 낙하, 4개 연결 승리 |
| ⬜ **오셀로** | 8×8 뒤집기, 유효 수 자동 표시·패스 처리 |
| 🃏 **인디언 포커** | 상대 카드로 심리전, 배팅·레이즈·폴드 |
| 🔴 **체커** | 강제 점프 룰, 연속 점프, 킹 승격 |

## 주요 기능

### 게임 플레이
- 실시간 1대1 대국 (Socket.io WebSocket)
- 색상 선택 (방장 선택)
- 제한 시간: 10분 / 30분 / 무제한 / 직접 설정 (1~120분)
- 재접속 지원 — UUID 토큰으로 새로고침 후 10분 이내 복귀
- 재대국 기능 (색상 자동 교체)
- 관전자 모드 (방 번호로 입장, 방장 승인 방식)

### 부가 기능
- 실시간 채팅 + 이모티콘 (👍😂😮😤🤝🔥💀♟)
- 대국 복기 (체스 전용, 키보드 ← → 지원)
- 사운드 효과 (Web Audio API, 외부 파일 없음)
- 관리자 대시보드 (`/admin.html`) — 서버 현황 실시간 모니터링

### 인프라
- Render.com 클라우드 배포 (고정 URL, 24시간 운영)
- Cloudflare Tunnel 로컬 모드 (PC에서 직접 실행 시)
- PWA manifest + Service Worker (홈 화면 추가 지원)
- 개인정보처리방침 페이지 (`/privacy.html`)

## 보안

| 항목 | 내용 |
|------|------|
| 서버 Rate Limit | create 5/60s, join 10/60s, reconnect 5/60s, move 30/10s, chat 20/10s |
| 기권 Rate Limit | 분당 최대 3회 |
| 무승부 제안 Rate Limit | 분당 최대 5회 |
| 버튼 클릭 보호 | 무승부: 5초 딜레이·3회 초과 시 60초 비활성화 |
| 기권 버튼 보호 | 3초 쿨다운 + 이중 전송 방지 |
| 인디언 포커 액션 | fold/call/raise 1.5초 debounce |
| 입력 검증 | 좌표 범위, gameType 허용 목록, chat 길이 제한 |
| 카드 정보 은닉 | 인디언 포커 내 카드는 개인 소켓으로만 발송 |
| 셧다운 키 | UUID, localhost 접속 시에만 노출, 파일 권한 0o600 |
