## 1. 클라우드 서버로 바로 플레이 (권장)

설치 없이 링크만으로 즉시 플레이 가능합니다.

### 서버 주소

```
https://board-game-online.onrender.com
```

### 플레이 방법

1. 위 주소를 브라우저에서 열기
2. 게임 선택 → 방 만들기
3. 생성된 링크를 카카오톡으로 친구에게 전송
4. 친구가 링크를 눌러 참가하면 게임 시작

> **참고**: 서버가 슬립 상태일 경우 첫 접속 시 30~60초 로딩이 발생할 수 있습니다.
> 페이지를 새로고침하면 정상 접속됩니다.

## 2. 로컬 서버 직접 실행 (개발자용)

코드를 수정하거나 로컬에서 테스트할 때 사용합니다.

### 사전 준비

| 소프트웨어 | 확인 방법 | 다운로드 |
|-----------|-----------|----------|
| **Node.js** (v18 이상) | 터미널에 `node -v` 입력 | https://nodejs.org |
| **Git** | 터미널에 `git --version` 입력 | https://git-scm.com |

### 설치 및 실행

```bash
# 1. 저장소 클론
git clone https://github.com/KulkulZa1/board_game_online.git
cd board_game_online

# 2. 의존성 설치
npm install

# 3. 서버 실행
node server.js
```

브라우저에서 `http://localhost:3000` 접속

### 외부 친구와 테스트 (Cloudflare Tunnel)

같은 WiFi가 아닌 외부 친구와 로컬 서버로 테스트할 때:

1. `cloudflared.exe` 를 프로젝트 폴더에 위치
   - 다운로드: https://github.com/cloudflare/cloudflared/releases/latest
   - `cloudflared-windows-amd64.exe` → `cloudflared.exe` 로 이름 변경
2. 별도 터미널에서 실행:
   ```bash
   cloudflared tunnel --url http://localhost:3000
   ```
3. 출력된 `https://xxxx.trycloudflare.com` URL을 친구에게 공유

> **주의**: 터미널 창을 닫으면 터널이 끊어집니다. 매번 새 URL이 생성됩니다.

## 3. 게임 방법 — 공통

### 방 만들기 (호스트)

1. 서버 주소 접속 후 게임 선택
2. **내 색상** 선택 (사목·인디언 포커는 자동 배정)
3. **제한 시간** 선택:
   - 10분 / 30분: 각자 해당 시간 내에 모든 수를 완료
   - 무제한: 시간 제한 없음
   - 직접 설정: 1~120분
4. **방 만들기** 클릭
5. 생성된 링크를 복사하여 친구에게 전송

### 방 참가 (게스트)

받은 링크를 클릭하면 자동으로 게임 시작

### 관전

방 번호를 알고 있으면 URL에 `?spectate=<방번호>` 를 추가하여 관전 입장 가능.
방장의 승인이 필요합니다.

### 게임 중 공통 기능

| 버튼 | 기능 |
|------|------|
| **기권** | 상대방 승리로 게임 종료 (3초 쿨다운 적용) |
| **무승부 제안** | 상대방에게 무승부 제안 전송 (체스 전용, 5초 딜레이·3회 초과 시 60초 비활성) |
| **💬 채팅** | 채팅창 열기/닫기 |
| **🔊 / 🔇** | 사운드 켜기/끄기 |

### 게임 종료 후

| 버튼 | 기능 |
|------|------|
| **재대국** | 같은 상대와 색상을 바꿔 다시 대국 |
| **새 게임** | 로비로 돌아가 새 방 생성 |

### 재접속

게임 중 새로고침이나 연결이 끊겨도 **10분 이내** 같은 링크로 재접속하면 게임이 이어집니다.

### 모바일 이용 시 주의사항

- 가로 모드 권장 (특히 체스, 오목)
- 채팅창을 열면 키보드가 올라와 화면이 좁아집니다. 채팅 후 닫기 버튼을 누르세요
- 체스 복기는 모바일에서도 지원합니다 (← → 버튼 사용)

## 4. 게임별 규칙

### ♟ 체스

- 이동할 기물을 **클릭** → 이동 가능한 칸 표시 → 목적지 클릭
- 폰이 상대 진영 끝에 도달하면 **승급 선택 화면** 자동 표시
- 게임 종료 후 **복기 모드** 진입 가능 (⏮◀▶⏭, 키보드 ← → 지원)
- 무승부 제안은 체스에서만 사용 가능

### ⬤ 오목 (렌주 룰)

- 빈 교차점을 클릭해 돌을 놓습니다
- **정확히 5개** 연결 시 승리 (6개 이상 장목은 무효)
- 흑이 선공

### 🔴 사목 (Connect Four)

- 상단 화살표 버튼 또는 열 클릭으로 돌 투하
- 중력이 적용됩니다 (빈 칸 있으면 아래로 떨어짐)
- **4개 연결** (가로·세로·대각선) 시 승리
- 호스트 = 빨강(선공), 게스트 = 노랑

### ⬜ 오셀로 (리버시)

- 상대 돌을 사이에 끼울 수 있는 칸에만 놓을 수 있습니다
- 유효 수 위치에 **점** 으로 표시됩니다
- 유효 수가 없으면 자동으로 **패스**됩니다
- 양측 모두 유효 수 없으면 돌 수 비교로 승패 결정
- 흑이 선공

### 🃏 인디언 포커

- 자기 카드는 이마에 대고 보지 않습니다 (화면에 "?" 로 표시)
- **상대방 카드**만 볼 수 있습니다
- 배팅 순서: 게스트 먼저 → 호스트 → 쇼다운
- 액션: **콜**(배팅 맞추기) / **레이즈**(5칩 추가, 최대 3회) / **폴드**(포기)
- 각 액션은 **1.5초에 1번**만 가능합니다 (연속 클릭 방지)
- **높은 숫자**가 이깁니다 (A < 2 < ... < K). 동점이면 호스트 승
- 칩이 앤티(5칩)보다 적으면 게임 종료

### 🔴 체커 (Checkers)

- 어두운 칸에서만 이동합니다
- 기본 이동: 앞 대각선 1칸 이동
- **점프**: 상대 말을 대각선으로 뛰어넘어 잡을 수 있습니다
  - 점프 가능하면 **반드시 해야 합니다** (강제 점프 룰)
  - 연속 점프 가능하면 계속 같은 말로 이동합니다
- 상대 진영 끝에 도달하면 **킹**(♛) 으로 승격 → 전후 방향 이동 가능
- 상대 말을 전멸시키거나 이동 불가 상태로 만들면 승리
- 호스트: 빨강(선공), 게스트: 검정

## 6. 문제 해결

### 첫 접속 시 로딩이 오래 걸린다

Render.com 무료 플랜은 15분 무활동 시 슬립 상태로 전환됩니다.
첫 요청 시 30~60초 로딩이 발생할 수 있습니다.

**해결**: 잠시 기다린 후 페이지를 새로고침하세요.

---

### 친구가 접속이 안 된다

1. 친구에게 전달한 URL이 `https://board-game-online.onrender.com` 으로 시작하는지 확인
2. 방 링크가 올바른지 확인 (`?room=...` 파라미터 포함)
3. 방이 이미 게임 중(active)이면 참가 불가

---

### 사운드가 나지 않는다

- 게임 화면의 🔇 버튼 클릭하여 🔊 로 변경
- 첫 수를 두거나 버튼을 클릭하면 오디오 활성화됩니다 (브라우저 자동재생 정책)

---

### 무승부 제안 버튼이 작동하지 않는다

- 연속 클릭 보호 기능이 있습니다:
  - 클릭 간 **5초 딜레이** 필요
  - 3회 초과 시 **60초 비활성화** (버튼에 "대기 중" 표시)
- 체스 게임에서만 사용 가능합니다

---

### 기권 버튼이 반응하지 않는다

- **3초 쿨다운**이 있습니다. 잠시 후 다시 클릭하세요
- 현재 게임 중(active)인지 확인하세요

---

### 체커에서 말이 움직이지 않는다

- 현재 자신의 차례인지 확인 (상단 상태 표시 확인)
- 점프 가능한 말이 있으면 그 말만 이동 가능합니다 (강제 점프)
- 연속 점프 중에는 같은 말만 계속 이동해야 합니다

---

### 인디언 포커에서 배팅 버튼이 안 보인다

- 자신의 배팅 차례가 아닐 때는 버튼이 숨겨집니다
- 방금 버튼을 눌렀다면 **1.5초 후** 다시 클릭하세요
