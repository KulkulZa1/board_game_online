## 목표

Android Play Store 제출을 위한 PWA 완성 + 소규모 버그 수정

---

## 예정 작업

### 필수 (Play Store 제출 조건)

| 항목 | 내용 |
|------|------|
| PWA 아이콘 3종 | `icon-192.png`, `icon-512.png`, `icon-512-maskable.png` 실제 이미지 |
| `assetlinks.json` 업데이트 | APK 서명 SHA-256 핑거프린트 입력 |
| TWA(Trusted Web Activity) 빌드 | Bubblewrap CLI로 Android APK/AAB 생성 |
| Play Store 제출 | 스크린샷·설명·개인정보처리방침 등록 |

### 버그 수정

| 항목 | 파일 | 내용 |
|------|------|------|
| iOS 긴 터치 메뉴 차단 | `public/css/game.css` | `-webkit-touch-callout: none` |
| rateLimits 메모리 정리 | `server.js` | 1시간마다 오래된 항목 삭제 |

### 선택 (여유 있을 경우)

- 채팅 이모티콘 추가
- 오목 힌트 표시 옵션
- 관리자 페이지 방 강제 종료 기능
