개인정보처리방침 Privacy Policy

보드게임 온라인은 사용자의 개인정보를 중요하게 생각하며, 아래와 같이 최소한의 원칙으로 서비스를 운영합니다.

1. 수집하는 개인정보

본 서비스는 개인정보를 수집하지 않습니다.

  • 회원가입 없음 — 이메일, 이름, 비밀번호 등 수집 없음
  • 로그인 없음 — 소셜 로그인 포함 인증 없음
  • 닉네임 없음 — 게임 내 이름 저장 없음

2. 서버에 임시 저장되는 정보

게임 진행 중 다음 정보가 서버 메모리에 일시적으로 저장됩니다:

  • 게임 방 ID (UUID 무작위 생성)
  • 게임 진행 상태 (기물 위치, 이동 기록, 남은 시간)
  • 익명 플레이어 토큰 (재접속용, 브라우저 sessionStorage에 저장)
  • 채팅 내역 (게임 종료 또는 방 삭제 시 즉시 삭제)

이 정보는 서버 재시작 또는 게임 종료 시 자동으로 삭제되며, 영구 데이터베이스에 저장되지 않습니다.

3. 쿠키 및 추적

본 서비스는 쿠키를 사용하지 않으며, 광고 추적 스크립트를 포함하지 않습니다. 플레이어 토큰은 브라우저의 sessionStorage에만 저장됩니다 (브라우저 탭 종료 시 자동 삭제).

4. 제3자 공유

수집된 정보(없음)를 제3자와 공유하지 않습니다. Cloudflare Tunnel을 통해 서비스를 제공할 경우 Cloudflare의 개인정보처리방침이 네트워크 수준에서 적용될 수 있습니다.

5. 아동 개인정보 보호

본 서비스는 13세 미만 아동으로부터 개인정보를 수집하지 않습니다. 만 13세 미만은 보호자의 동의하에 이용하시기 바랍니다.

6. 문의

개인정보 관련 문의 사항이 있으시면 앱 스토어 개발자 페이지를 통해 연락해 주세요.
