# 보드게임 온라인

카카오톡 링크 공유로 친구와 실시간 1대1 보드게임을 즐길 수 있는 웹 앱입니다.

## 지원 게임

| 게임 | 설명 |
|------|------|
| ♟ **체스** | 고전 전략 게임. 폰 승급, 캐슬링, 앙파상 완전 지원 |
| ⬤ **오목** | 15×15 렌주 룰 (정확히 5개 연결, 장목은 무효) |
| 🔴 **사목** | 7×6 중력 낙하, 4개 연결 승리 |
| ⬜ **오셀로** | 8×8 뒤집기 게임. 유효 수 표시, 자동 패스 처리 |
| 🃏 **인디언 포커** | 상대 카드만 보는 심리전. 배팅·레이즈·폴드 |
| 🔴 **체커** | 8×8 점프 이동, 강제 점프 룰, 킹 승격 |

## 실행 방법

### 1단계: Node.js 설치 (처음 1회만)

https://nodejs.org 에서 LTS 버전 다운로드 후 설치

또는 터미널에서:
```
winget install OpenJS.NodeJS.LTS
```

### 2단계: 서버 실행

`start.bat` 파일을 더블클릭하거나, 터미널에서:

```bash
cd chess_online
npm install
node server.js
```

### 3단계: 접속

브라우저에서 `http://localhost:3000` 접속

### 4단계: 친구와 대국

1. 게임 선택 → 색상/시간 설정 → 방 만들기 → 링크 생성
2. 카카오톡으로 링크 전송
3. 친구가 링크를 눌러 참가

## 외부(인터넷)에서 접속하려면

**Cloudflare Tunnel (내장)**: `start-public.bat` 더블클릭
→ 자동으로 공개 URL 생성 및 표시

같은 WiFi라면 내 IP로 접속 가능 (`start.bat` 실행 후 표시되는 LAN 주소).

## 카카오톡 공유 버튼 활성화

`public/js/lobby.js` 파일에서 `YOUR_KAKAO_APP_KEY`를 실제 앱 키로 교체:

1. https://developers.kakao.com 에서 앱 생성
2. JavaScript 앱 키 복사
3. `lobby.js` 상단 `const KAKAO_APP_KEY = 'YOUR_KAKAO_APP_KEY';` 수정

카카오 앱 키 없이도 **링크 복사** 버튼으로 공유 가능합니다.

## 주요 기능

- **6가지 게임**: 체스·오목·사목·오셀로·인디언포커·체커
- 색상 선택 (방장이 선택)
- 제한 시간: 10분 / 30분 / 무제한 / 직접 설정
- 실시간 채팅 + 이모티콘 (👍😂😮😤🤝🔥💀♟)
- 대국 복기 (체스 전용, 키보드 방향키 지원)
- 재접속 지원 (새로고침해도 게임 유지)
- 재대국 기능
- 관전자 모드 (방 번호로 입장)
- PC + Android 모바일 지원

## 환경

- Node.js 18 이상
- 포트: 3000 (기본값)
- 외부 접속: cloudflared.exe 동봉 (Cloudflare Tunnel)
