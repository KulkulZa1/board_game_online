# 보드게임 온라인

카카오톡 링크 공유로 친구와 실시간 1대1 보드게임을 즐길 수 있는 웹 앱입니다.

**배포 URL**: https://board-game-online.onrender.com
**버전**: v1.4.0 | **GitHub**: https://github.com/KulkulZa1/board_game_online

---

## 지원 게임 (12종)

| 게임 | 설명 |
|------|------|
| ♟ **체스** | 고전 전략 게임. 폰 승급·캐슬링·앙파상 완전 지원 |
| ⬤ **오목** | 15×15 렌주 룰 (정확히 5개 연결, 장목은 무효) |
| 🔴 **사목** | 7×6 중력 낙하, 4개 연결 승리 |
| ⬜ **오셀로** | 8×8 뒤집기. 유효 수 표시, 자동 패스 처리 |
| 🃏 **인디언 포커** | 상대 카드만 보는 심리전. 배팅·레이즈·폴드 |
| 🔴 **체커** | 8×8 점프 이동, 강제 점프 룰, 킹 승격 |
| 🍎 **사과게임** | 17×10 격자, 합이 10이 되는 사각형 선택·제거 |
| 🚢 **배틀십** | 10×10 해전, 함선 배치 후 교대 공격 |
| 🎲 **백가몬** | 24포인트 보드, 주사위 2개, 바·탈출 완전 구현 |
| ♠ **텍사스 홀덤** | 헤즈업 포커, 블라인드·4라운드 베팅·핸드 평가 |
| 📦 **도트앤박스** | 5×5 격자, 선 연결로 박스 완성, 보너스 턴 |
| 🪴 **만칼라** | 14구멍 보드, 씨앗 배분·캡처 전략 |

## 바로 플레이

별도 설치 없이 아래 URL로 접속하세요:

```
https://board-game-online.onrender.com
```

1. 게임 선택 → 방 만들기
2. 생성된 링크를 카카오톡으로 친구에게 전송
3. 친구가 링크 클릭 → 즉시 대국 시작

---

## 개발 환경 설정

```bash
git clone https://github.com/KulkulZa1/board_game_online.git
cd board_game_online
npm install
node server.js
# → http://localhost:3000
```

**Node.js 18 이상** 필요

---

## 주요 기능

- 12가지 게임 지원
- **혼자하기 (vs AI)**: 12게임 모두 AI 대국 지원
- 오목·사목 보드 크기 선택 (멀티·솔로 모두)
- 색상 선택 + 제한 시간 설정 (10분/30분/무제한/직접 설정)
- 실시간 채팅 + 이모티콘 (👍😂😮😤🤝🔥💀♟)
- 대국 복기 (체스 전용, 키보드 방향키 지원)
- 재접속 지원 (새로고침해도 게임 유지)
- 재대국 기능 (색상 자동 교체)
- 관전자 모드
- 개인 전적 기록 (게임별 승/패/무 통계)
- PC + Android 모바일 지원
- PWA (홈 화면 추가 지원)
